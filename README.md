## Descrição
<img width="800" height="400" alt="messier_logo" src="https://github.com/user-attachments/assets/7ee1e891-317b-4e71-8dfb-e9cc3249598b" />

  O projeto consiste no desenvolvimento de um aplicativo desktop para gerenciar assinaturas de jogos educacionais voltados a escolas de ensino fundamental e médio. A solução permitirá o cadastro e controle de escolas, pacotes de jogos e acessos, incluindo validação de IP autorizado, limite de uso mensal e registro de atividades. Além disso, o sistema contará com relatórios de consumo e organização em banco de dados, aplicando conceitos de programação, modelagem de sistemas e redes.
