# Sistema Desktop para Gestão de Assinaturas de Jogos Educacionais

Projeto desenvolvido para o Projeto Integrador do curso de Análise e Desenvolvimento de Sistemas.

Aplicação desktop voltada ao gerenciamento de assinaturas de jogos educacionais para escolas, com controle de pacotes, validação de acesso por IP e monitoramento de consumo mensal.

## Objetivo

Desenvolver uma aplicação integrada a banco de dados capaz de gerenciar escolas, pacotes e jogos, garantindo o controle de acessos e validação de regras como limite mensal e autorização por IP.
